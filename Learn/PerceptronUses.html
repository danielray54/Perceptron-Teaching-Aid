How are Artificial Neural Networks Used Today?

Artificial Neural Networks can be applied to a vast array of fields, below I will detail a sample of the various areas that ANN's can be used in:

  • Aerospace − Autopilot aircrafts, aircraft fault detection.
  • Automotive − Automobile guidance systems.
  • Military − Weapon orientation and steering, target tracking, object discrimination, facial recognition, signal/image identification.
  • Electronics − Code sequence prediction, IC chip layout, chip failure analysis, machine vision, voice synthesis.
  • Financial − Real estate appraisal, loan advisor, mortgage screening, corporate bond rating, portfolio trading program, corporate financial analysis, currency value prediction, document readers, credit application evaluators.
  • Industrial − Manufacturing process control, product design and analysis, quality inspection systems, welding quality analysis, paper quality prediction, chemical product design analysis, dynamic modeling of chemical process systems, machine maintenance analysis, project bidding, planning, and management.
  • Medical − Cancer cell analysis, EEG and ECG analysis, prosthetic design, transplant time optimizer.
  • Speech − Speech recognition, speech classification, text to speech conversion.
  • Telecommunications − Image and data compression, automated information services, real-time spoken language translation.
  • Transportation − Truck Brake system diagnosis, vehicle scheduling, routing systems.
  • Software − Pattern Recognition in facial recognition, optical character recognition, etc.
  • Time Series Prediction − ANNs are used to make predictions on stocks and natural calamities.
  • Signal Processing − Neural networks can be trained to process an audio signal and filter it appropriately in the hearing aids.
  • Control − ANNs are often used to make steering decisions of physical vehicles. Anomaly Detection − As ANNs are expert at recognizing patterns, they can also be trained to generate an output when something unusual occurs that misfits the pattern.

How Can the Perceptron be Used?

Looking at the list of application of Artificial Neural Networks, you can see that one main use is Pattern Recognition. The Perceptron can be used to classify data by recoginsing patterns within that data.
If we look at the Demonstration below, there is a coordinate plane where a number of points are added in order to train the perceptron to recognise a pattern within the coordinate plane in order to classify new data points according to that pattern. The training data will be the x and y coordinates and the target output will be dependant on the relationship between x and y (If X is greater than Y, the training point part of Class 1 otherwise the training point is part of Class -1).



Instructions

Step 1: Enter the Number of Training Sets into the Text Field (e.g. 2000)
Step 2: Enter the Learning Rate using the Slider
Step 3: Click the Show Points button to display the training points on screen
Notice: the dark points are part of one Class, whilst the light points are part of another Class
Step 4: Click the Train button to begin the training phase
Notice: The points with green detailing represent the perceptron correctly prediciting that point, whilst red represents incorrect predicitions for that point.
Step 5: If you wish to predict a new point, with an unknown output. Press the Stop button and enter the Inputs for the point, then pressing the Predict button.
If you wish to reload the demonstration at any point, press the Reload button.


Fisher's Iris Data Set

We can now apply this functionality to real world data. Ronald Fisher, a Statistician and Biologist, collected data on three types of Iris Flower species; Iris setosa, Iris virginica and Iris versicolor.

Four features were measured from each sample: the length and the width of the sepals and petals, in centimetres. Due to the nature of the Perceptron being a binary classifier, we can implement the All vs One approach to the data set in order to classify whether an Iris Species is either Setosa or Virginica/ Versicolor. Below is a sample of the data set used to train the Perceptron:
